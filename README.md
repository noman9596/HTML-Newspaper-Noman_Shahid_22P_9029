Project Description
This project recreates the front page of the old Urdu newspaper Kar-e-Jahan using only HTML.
A table-based layout and semantic HTML5 tags were used to match the traditional multi-column newspaper design.

Features Implemented

Semantic HTML5 elements
Two-column table layout
Inline styling for structure
Images with alt attributes
News briefs using lists

Challenges Faced

Creating a structured newspaper layout without CSS was challenging.
Tables and inline formatting were carefully used to maintain proper alignment and layout.
